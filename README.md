# dojo-poll
Node poll app using sockets

## Technologies Used
* node.js
* express.js
* socket.io
* jQuery
* jwt
* chart.js

## How to get it running

1. Install npm and node
```shell
$ sudo apt-get update
$ sudo apt-get install node
$ sudo apt-get install npm
```

2. Install node packages
```shell
$ npm install
```

3. Start the server
```shell
$ node server.js
```
